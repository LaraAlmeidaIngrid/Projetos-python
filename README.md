# Projetos da faculdade

Esse repositório foi desenvolvido durante o meu primeiro semestre no curso de Ciência da Computação, aqui temos projetos bem simples que marcam meu início no aprendizado em Python e Lógica de Programação.

---

# Visão Geral 

## 1. Simulador de Viagens

Esse é um projeto em que o usuário pode decidir para qual planeta quer viajar, consegui utilizar a manipulação de textos em python para analisar o nome do usuário, além de verificar as condições específicas para a viagem de cada pessoa utilizando as estruturas condicionais. Por fim, fiz o teste de cores no terminal no intuito de produzir um cartão de viagens colorido e atrativo.

## 2. Caixa Eletrônico

Utilizando as estruturas de repetição, consegui desenvolver um caixa bem simples para que o usuário verificasse o saldo disponível, além de retirar ou depositar algum valor. Me atentei para utilizar as estruturas condicionais no tratamento de erros, evitando que o usuário conseguisse quebrar o sistema.

## 3. Sistema da Lanchonete

Nesse projeto foi disponibilizado um cardápio em que por meio das estruturas de repetição e de controle de fluxo (match case) o usuário fazia a escolha do seu pedido, a quantidade de cada item escolhido, finalizava e recebia o valor total. Além disso, foram utilizadas outras estruturas para garantir o funcionamento pleno do sistema.

## 4. Sistema da Prefeitura 

Esse foi um trabalho avaliativo que desenvolvi em conjunto com outra colega de classe. Desenvolvemos um pequeno sistema para a prefeitura da nossa cidade que facilitava o acesso aos serviços da Secretaria de Obras, Frotas e de Educação. As estruturas exigidas para que esse programa fosse desenvolvido foram as condicionais e de controle de fluxo.
